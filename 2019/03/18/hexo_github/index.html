利用hexo+github搭建博客

工具 blog搭建 2019/03/18

如果你想搭建一个博客，又不知该从何入手，可以阅读这篇文章，本文是利用hexo和github来搭建个人博客，小白搭建博客必备。

ps:本人使用的是win10操作系统，其他操作系统可能会略有不同

一、简介

1. 搭建博客常见的3种方案

1. Github pages+hexo+markdown
2. Github pages+jekyll+markdown
3. vps主机+wordpress+插件

这里我选择github+hexo+markdown方式搭建

2. git简介

Git 是 Linus 为了帮助管理 Linux 内核而开发的一个开源免费的版本控制工具，与CVS、SVN这些集中式的版本控制系统不同，Git是分布式版本控制系统。虽然分布式版本控制系统除了Git外，还有促使Git诞生的BitKeeper，以及类似Git的Mercurial和Bazaar等，但是最快、最简单也最流行的依然是Git！

3. node.js简介

在Netscape设计出JavaScript后不久，JavaScript已经成为了前端开发的唯一标准，此后Google基于WebKit内核推出了Chrome浏览器，并自己开发了一个高性能JavaScript引擎—V8，后来一个名叫Ryan Dahl的外国人在此基础上，于2009年推出了基于JavaScript语言和V8引擎的开源Web服务器项目，命名为Node.js。Node第一次把JavaScript带入到后端服务器开发，它让 JavaScript 成为了与PHP、Python、Perl、Ruby 等服务端语言平起平坐的脚本语言

4. hexo简介

Hexo 是一个基于Node.js的快速、简洁、高效的博客框架，Hexo 需要Node.js环境的支持，使用Markdown作为渲染引擎解析文章，结合丰富的主题，可快速生成绚丽的静态网页

node.js官网下载地址：https://nodejs.org/en/

安装过程默认推荐即可，这里不作过多描述

三、hexo远端部署github pages

1. 注册github账号

github官网：https://github.com/
按要求填写个人信息即可

2. 创建仓库

登录github，点击new repository新建仓库

repository name的格式为：yourname.github.io

把yourname替换成个人github的注册名

最后勾选Initialize this repository with a README,点击creat repository
